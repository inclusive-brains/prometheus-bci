About this interface

This second version of the Prometheus BCI platform was designed to enable Pierre Landerretche to control his exoskeleton arm and carry the Olympic flame on July 24, 2024, in Versailles.

This section allows adjustments to motor command control settings and cognitive metrics. Remember to update the probability accumulator settings in the "Mind Control calibration" interface (after the calibration). In this example we chose to use only one mental command but you could use two commands (left arm vs right arm for example).
